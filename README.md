# postgresql

## Enter Databse CLI

```postgresql
psql postgres
# or
psql databseNm
```


## Show Databases

```postgresql
\l
```

## Change Current Databse

```postgresql
\c database_name
```

## Show Tables

```postgresql
\dt
```

## Show Table Schema

```postgresql
\d table_name
```

## Drop Database

```postgresql
DROP DATABASE databaseName
```
